📚 Manual Conversion Guide

For best results with large, encrypted, or complex PST exports, use one of these proven tools to convert PST → EML, then load the EML files into the viewer.

Microsoft Outlook
Windows • Mac
Open PST via File → Open & Export. Select emails → drag to desktop as .eml or save as .msg.
Aid4Mail
Windows • Free tier
GUI tool with drag-and-drop PST → EML batch conversion. Very reliable for eDiscovery exports.
readpst
Linux • Mac • Free
Command-line tool (part of libpst). The fastest option for bulk conversion of large PST files.
Thunderbird
Windows • Mac • Linux
Free email client. Import PST via ImportExportTools NG extension, then export all as EML.
Step-by-step: readpst (fastest for bulk)
1
Install readpst

Mac: brew install libpst

Ubuntu / Debian: sudo apt install pst-utils

2
Run the conversion

readpst -e -o ./output_folder your_file.pst

The -e flag saves each message as an individual .eml file. -o sets where they are saved.

3
Open in the viewer

Go back to the Chat Viewer and drag all the EML files in from your output folder.
